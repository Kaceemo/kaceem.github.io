Privacy Policy

By inviting and/or using the bot in your server you agree to the following Privacy Policy

  • Storing certain data is required for the functionality of the bot (Will be mentioned in the next point). We are not allowed to use the data outside of this application and it cannot be shared to any 3rd-party services

  • Stored information.
    • User ID: Whenever a discord user creates a new streak, his user ID is added to our database.
    • Server ID: Whenever a streak is created on a server, the server ID is added to our database.

  • For what reason is data stored?
    • User ID: Required so that each user can have his own unique streak assigned to him.
    • Server ID: Required so that each server can have its own unique streaks separated from others along with a leaderboard system. Also allows users to have separate streaks on different servers.

  • Data Removal: The data removal process is automatic in the following cases:
    • Once the bot is kicked from a server, all data assigned to that server is deleted from the database (streaks) which includes the server ID and all users IDs on that server.
    • Once a user resets his streak and/or a user's streak is forcesfully reset by an admin, the said user's ID is deleted from the database.
    • Once the /leaderboard command is ran, the bot automatically scans all streaks to filter out the users that are no longer in the server and deletes all their user IDs. (this mechanism was put in place to make sure user IDs do not stay stored after a user leaves the server)
    • In case of any doubt you may join our support server and a developer will be available to manually confirm that all of your data is deleted.
